# Ticket Classifier

Ein Python-Projekt zur automatischen Klassifizierung von Helpdesk- und Security-Tickets mit Machine Learning.

## Funktion
Das Modell analysiert Ticket-Texte und ordnet sie einer Kategorie zu:
- Password Issue
- Network Issue
- Security Incident
- Software Issue

## Start
```bash
pip install pandas scikit-learn
python main.py
